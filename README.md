# AI vs Human Data Analysis Project

## Project Overview
This project performs a comparative analysis between AI-generated and Human-generated content. It examines 20,000 data points across various topics such as coding algorithms, finance, health, and tech to identify patterns in complexity, word count, and structure.

## Key Insights
- **Dataset Size:** 20,000 records (Approx. 50% AI, 50% Human).
- **Models Analyzed:** GPT-4, Claude, and Gemini.
- **Top Topics:** Sorting algorithms, Graph traversal, Finance, and Health.
- **Metrics:** Complexity scores, word counts, and multi-line code analysis.

## Tools Used
- **Power BI:** For interactive data visualization and dashboarding.
- **Excel/CSV:** For data storage and initial cleaning.
- **Python (Optional):** Used for data processing.

## How to Use
1. Download the `.pbix` file from the `/Report` folder.
2. Open it in Power BI Desktop to interact with the filters.
